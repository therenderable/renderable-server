# Renderable Server

Renderable web server.

## Prerequisites

* [Docker Engine >=19.03.12](https://docs.docker.com/engine)
* [Docker Compose >=1.27.2](https://docs.docker.com/compose)

## Installation

Build and run services:

```
docker-compose up -d
```
